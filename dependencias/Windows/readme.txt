/include
Contém o header do GLUT para inserir no projeto.
Se desejar apenas insira na pasta include do IDE
Exemplo no Code::Blocks:
C:\Program Files (x86)\CodeBlocks\MinGW\include
*******************************************************************
/lib
Contém a lib já compilada do GLUT para inserir no projeto.
Se desejar apenas insira na pasta include do IDE
Exemplo no Code::Blocks:
C:\Program Files (x86)\CodeBlocks\MinGW\lib
*******************************************************************
/variavel de ambiente
Insira ou crie um diretório de variável de ambiente para o "glut32.dll".

Se não sabe o que é isso apenas copie "glut32.dll" para
C:\Windows\System
ou
C:\Windows\System32
*******************************************************************
